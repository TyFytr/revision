What are Classes and ID's? (and how do they differ)

What are classes?

  • Classes are a way of targeting specific headers, paragraphs, div's, etc, while still maintaning general style rules.
  • Each class can be used multiple times.
  • Each class can have multiple rules.

What are ID's?

  • ID's are used when targeting VERY specific style rules within headers, paragraths, etc.
  • ID's can only be used once
  • ID's should be used sparingly to reduce redundant coding

What are HTML tags?

HTML tages are hidden keywords within a webpage that define how your browser must format and display the content.

Examples-

  • The "body" tag indicates all the content that should be displayed
  • The "h1" tag indicates a header style number 1
  • The "strong" tag indicates all content should be bold
  • The "style" tag indicates the syle rules for the seclected elements

What are attridutes?

Attributes provide additional information about the HTML tags/elements

Examples-

  • The href indicates that the tag/elements is a link and the address of that link
  • width and height attributes define the specific styling rules of that tag/element
  • The scr attribute dines the source of an image for the tag/element

What are HTML selectors, properties and values?

HTML "Selectors" are the names of the HTML tags that are define specific type of content/element.

For Example:

  • body: this tag specifies all content/elements within the "body" tags
  • h1: this tag specifies all content/elements within the "heading 1" tags
  • p: this tag spefcifies all the content/elements within the "paragrph tags"

Each selector has "properties", and these "properties are placed inside curly brackets

These "properties" are used to change the style and formate of the content/elements within the "HTML selector" tags.

For example:

  • color: with will define the color of the element
  • font-family: this will define the the font style of all text within the elements
  • font-size: this will define the size of the text within the element

Each of the "properties" will have values attached. These values specify the how the output/input of the

element will be displayed/formatted, etc. These values come after the properties and a colon with a

semi-colon seperating each property and value

For example:

  • font-size(this been the property) : 14px(this been the value);
  • font-family(this been the property) : arial(this been the value);

Applying CSS

CSS stands for Cascading style sheets. CSS is what gives your HTML document its styling/presentation.

There are three ways CSS can be applied to an HTML document:

  1. Inline: This is when the CSS is applied directly to the HTML tag/element. The values will apply only to that tag/element
  2. Internal: This is when the CSS is applied to the "head" of the HTML document. The values will apply to the whole document
  3. External: This is when a separtae CSS file is created. The values can be applied to multiple pages/documents,etc.

    That sheet must be linked to the HTML pages/documents that will use the CSS
