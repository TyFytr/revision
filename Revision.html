What are Classes and ID's? (and how do they differ)

What are classes?

  • Classes are a way of targeting specific headers, paragraphs, div's, etc, while still maintaing general style rules.
  • Each class can be used multile times.
  • Each class can have multile rules.

What are ID's?

  • ID's are used when targeting VERY specific style rules within headers, paragraths, etc.
  • ID's can only be used once
  • ID's should be used sparingly to reduce redundant coding

Whata are HTML tags?

HTML tages are hidden keywords within a webpage taht define how your browser must format and display the content.

Examples-

  • The "body" tag indicates all the content that should be displayed
  • The "h1" tag indicates a header style number 1
  • The "strong" tag indicates all content should be bold
  • "style" tag indicates the syle rules for the seclected elements
